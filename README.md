# HR Data Analytics Dashboard (Power BI Project)

## Project Overview
This project analyzes employee attrition and workforce trends using Power BI.

The dashboard helps HR teams understand:

- Employee attrition patterns
- Workforce demographics
- Department performance
- Key factors influencing employee turnover

## Tools Used
- Power BI
- Power Query
- DAX
- Kaggle Dataset

## Dashboard Pages
1. HR Overview Dashboard
2. Employee Demographics
3. Attrition Factors

## Key Insights
- Early career employees have higher attrition
- Lower salary groups experience more turnover
- Sales and R&D departments show higher attrition
- Overtime work increases attrition
